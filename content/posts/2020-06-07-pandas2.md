# Pandas 2
pandas를 사용하다 보니 성능 문제가 생기게 되어 블로깅 해보게 되었다.
pandas는 dataframe에 있는 내용들을 연산시 cpu의 1개의 코어만 사용하는 문제가 있다. 그러다 보니 나머지 코어들은 놀게되어 효율이 떨어 지게된다.
그래서 modin을 사용하여 병렬처리하여 효율을 높일 수 있다고 한다.
나는 적용해보았으나 생각보다 성능개선이 이루어 졌다고 느낄 수 없었으나 확실히 병렬 처리가 된다는 내용을 어느정도 확인해서 기록해 두겠다.


1. [mondin document](https://modin.readthedocs.io/en/latest/installation.html) 공식문서에 들어가서 설치법을 확인한다.

2. modin의 안정버전을 다운 받는다.
    ```
    pip install -U modin
    ```

3. modin은 그자체로만 작동하지 않으므로 추가적인 라이브러리를 설치해 주어야한다. 모두 다 설치해도되고 사용하기 원하는 것만 설치해도 된다.
    ```bash
    pip install dask
    pip install ray

    pip install "modin[ray]" # Install Modin dependencies and Ray to run on Ray
    pip install "modin[dask]" # Install Modin dependencies and Dask to run on Dask
    pip install "modin[all]" # Install all of the above
    ```

4. python에서 import하기
    ```bash
    # ray로 사용하는 법
    import ray
    ray.init(num_cpus=4)
    import modin.pandas as pd

    #dask로 사용하기
    import os
    os.environ["MODIN_CPUS"] = "4"
    import modin.pandas as pd
    ```

5. import 했다면 기존 pandas 처럼 사용하면된다.
