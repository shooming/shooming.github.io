# CSV 데이터 넣기
크롤링 혹은 무언가 데이터가 생성되어 SCV파일로 만들어져 있는 상태라면 데이터베이스에 자료를 넣어야 하는 상황이 오게 된다.  어떤식으로 자료를 넣으면 될까?

그 과정은 아래와 같다.

```python
import csv
import os
import django
import sys

os.chdir(".")
print("Current dir=", end=""), print(os.getcwd())

BASE_DIR = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
print("BASE_DIR=", end=""), print(BASE_DIR)

sys.path.append(BASE_DIR)

os.environ.setdefault("DJANGO_SETTINGS_MODULE", "project1.settings")
django.setup()

# 위쪽으로는 파이썬 쉘을 매번 실행하지 않고 파일로 만들어 빠르게 데이터를 넣기위해 파이썬쉘 실행하는 내용이다.
------------------------------------------------------------
# 아래는 데이터를 넣는 과정이다.

from car.models import *

# model
CSV_PATH = './CSV/model.csv'    #[1]

with open(CSV_PATH, newline='') as csvfile:
    data_reader = csv.DictReader(csvfile) #[2]
 
    for row in data_reader:        #[3]
        Model.objects.create(      #[4]
            name = row['model']    #[5]
        )
```
[1] CSV파일을 열 위치이다.

[2] csv의 파일을 dictionary 형태로 읽어 들인다는 뜻이다. 물론 그냥 Reader을 쓰면 리스트 형태로 불러온다.

[3] dictionary에 들은 내용을 하나씩 꺼낸다.

[4] row로 한줄 씩 저장한다.
