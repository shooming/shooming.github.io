```bash
myslq -h (RDS 주소) -u root -p
```
해당 RDS 주소로 접속이 가능하다.

```bash
mysqldump -u root -p (데이터베이스 이름) > (저장할 이름).sql
```
데이터 베이스 덤프를 하는 작업으로 내 기존 데이터베이스에서 내보내기하는 기능이다.

```bash
mysql -h (RDS 주소) -u root -p (데이터베이스 이름) < (덤프파일 이름).sql
```
<, > 데이터를 넣을 방향을 나타낸다.
그리고 데이터 밀어넣을 시 데이터베이스안에 데이터베이스가 생성되어있어야한다.
