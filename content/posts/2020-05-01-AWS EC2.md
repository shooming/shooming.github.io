gunicorn
장고 웹어플 돌려주는 프로그램

```
pip install gunicorn

gunicorn --bind=0.0.0.0:8000 장고웹어플.wsgi / 그냥 실행

nohup gunicorn --bind=0.0.0.0:8000 장고웹어플.wsgi & / & 백그라운드에서 실행하라는 뜻 
```
